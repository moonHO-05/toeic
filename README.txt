■ PC에서 여는 방법
  · 이 폴더의 index.html 을 더블클릭 → 크롬/엣지로 열림
  · 인터넷 없이도 작동합니다 (글꼴만 온라인일 때 더 예쁘게 나와요)

■ 📱 폰에서 쓰는 방법  (두 가지 중 택1)

  [방법 A] 같은 와이파이로 바로 열기 — 집에서 제일 빠름
    1) 이 폴더의 "폰에서_열기.bat" 을 더블클릭
    2) 검은 창에 뜨는 주소(http://192.168.x.x:5522)를
       폰 브라우저 주소창에 그대로 입력
    3) 끝나면 검은 창을 닫으면 서버가 꺼집니다.
    ※ 폰과 PC가 같은 와이파이여야 함 / PC가 켜져 있어야 함

  [방법 B] 인터넷에 올려서 어디서나 쓰기 — 지하철·학교에서도 OK
    1) 크롬에서  app.netlify.com/drop  접속
    2) 이 "토익단어장" 폴더를 통째로 화면에 드래그&드롭
    3) 뜨는 주소(예: https://xxxx.netlify.app)를 폰에서 열기
    4) 폰 브라우저 메뉴 → "홈 화면에 추가" 하면
       앱 아이콘(영단)이 생기고, 전체화면 앱처럼 켜집니다.
       (한 번 열면 인터넷 없이도 실행됨)

■ 구성 파일
  - index.html         화면 뼈대 + 앱/오프라인 설정
  - style.css          디자인
  - sound.js           효과음(코드로 소리 생성)
  - app.js             기능(목록/퀴즈/채점/저장)
  - words.js           단어 데이터 (ETS.xlsx 자동 추출, 3,155개)
  - manifest.json / sw.js / icon-*.png   폰 앱 설치·오프라인용
  - 폰에서_열기.bat     같은 와이파이로 여는 실행 파일

■ 사용법
  · 홈에서 DAY(1~30) 선택
  · [단어 목록] : 검색 + "뜻 가리고 셀프테스트"(뜻 흐리게→클릭 시 보임)
      - 단어의 ☆ 를 누르면 즐겨찾기(★). "☆ 즐겨찾기만" 으로 걸러 보기
  · [뜻 맞히기 퀴즈] : 영어 단어 → 한국어 뜻 4지선다
      - 한 세트 = 20문제, 순서는 매번 랜덤 (하루 약 100단어 → 5~7세트)
      - "← 이전 / 다음 →" 로 문제 넘나들기 (숫자키 1~4, ←/→, Enter 도 됨)
      - 카드의 ☆ 로 그 단어를 즐겨찾기에 담기
      - 🔊/🔇 버튼으로 효과음(정답/오답/완료) 켜고 끄기
      - 다 풀면 점수 + 틀린 단어를 보여줌(거기서도 ☆ 로 담기)
  · [즐겨찾기 복습] : 홈 위쪽 ★배너 → 별표한 단어만 모아서 퀴즈

■ 즐겨찾기
  · 별표(★)한 단어는 같은 브라우저에 저장됨 (localStorage: ets_favorites)
  · 진척도/점수 자동 저장 기능은 없음 → 복습할 단어는 ★ 로 직접 관리
  · 초기화: 개발자도구(F12) 콘솔에서
        localStorage.removeItem('ets_favorites')

■ 데이터 검수 메모
  · 뜻은 전반적으로 정확. 오타 1건만 수정:
    DAY 11 minimize 뜻의 "mazimize" → "maximize"
  · 괄호 속 (=동의어) (↔반의어) (to/with) 표기는 원문 그대로 유지
